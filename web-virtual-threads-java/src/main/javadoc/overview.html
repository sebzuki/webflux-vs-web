Ce module est monuments-keycloak.Il comprend :
  • Un fichier pom.xml contenant des dépendances génériques.
  • Un serveur HTTP intégré par Spring Boot (cf. fichier application.properties)
  • L'accès sécurisé par Spring Security est désactivé par défaut (cf. fichiers application.properties et SecurityConfiguration.java)
  • L'enrichissement des logs des services REST (durée, paramètres, etc.) par programmation orientée aspects (Spring AOP).
  • Un exemple de service REST avec service objet et DAO associée.
  • Un test unitaire correspondant au service REST d'exemple.
